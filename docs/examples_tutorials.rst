Examples & Tutorials
====================

MagBotSim offers the possibility to develop research-specific environments with a MagLev system.
These examples and tutorials will show how to use MagBotSim to build custom environments, to manually control movers etc.
Please use one of our :ref:`environments` as more extensive examples of how to use MagBotSim, especially with a RL API.
